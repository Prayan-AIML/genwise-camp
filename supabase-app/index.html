⚙︎ Enable live sync across devices (optional)
  1. Go to console.firebase.google.com → create a free project.
  2. Add a Web app; copy the firebaseConfig object.
  3. In Build → Realtime Database, create a database (test mode).
  4. Paste your config into the FIREBASE_CONFIG block near the top of this file's <script>.
  5. Save & reopen. Everyone using the same room code is now live.

Until then the hub runs in demo mode (saved in this browser; syncs across tabs on this device).
